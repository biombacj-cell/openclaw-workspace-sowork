# Atlas Skills Hub: 1000+ Agent Skills

This is ATLAS's unified agent skills hub - a curated collection of 1000+ agent skills compatible with Claude Code, Codex, Gemini CLI, Cursor, GitHub Copilot, OpenCode, Windsurf, and more.

**Attribution**: This skills collection is based on [VoltAgent/awesome-agent-skills](https://github.com/VoltAgent/awesome-agent-skills), the most comprehensive and actively maintained agent skills repository. We've integrated it into ATLAS to provide you with a complete, enterprise-grade AI learning system with thousands of pre-built skills.

                                                                                                                                                                                                              ## Using Skills with ATLAS
                                                                                                                                                                                                              
                                                                                                                                                                                                              Skills integrate seamlessly with ATLAS's self-learning system:
                                                                                                                                                                                                              
                                                                                                                                                                                                              ### 1. **Load a Skill**
                                                                                                                                                                                                              ```bash
                                                                                                                                                                                                              # Claude Code / Codex / Cursor / Gemini CLI
                                                                                                                                                                                                              # Skills are auto-loaded from .claude/skills/, .agents/skills/, .cursor/skills/, etc.
                                                                                                                                                                                                              ```
                                                                                                                                                                                                              
                                                                                                                                                                                                              ### 2. **Learn from Skill Execution**
                                                                                                                                                                                                              ATLAS's `HermesAdapter` automatically observes:
                                                                                                                                                                                                              - Which skills succeed/fail per market
                                                                                                                                                                                                              - Skill patterns and confidence scores
                                                                                                                                                                                                              - - Cross-market best practices
                                                                                                                                                                                                                - - Adaptive rule generation
                                                                                                                                                                                                                 
                                                                                                                                                                                                                  - ### 3. **Export Learned Patterns**
                                                                                                                                                                                                                  - ```python
                                                                                                                                                                                                                    from atlas import HermesAdapter, MarketRouter

                                                                                                                                                                                                                    router = MarketRouter()
                                                                                                                                                                                                                    adapter = HermesAdapter(market_router=router)

                                                                                                                                                                                                                    # After skill execution
                                                                                                                                                                                                                    recommendations = adapter.get_adaptive_recommendations("us-en", context={})
                                                                                                                                                                                                                    adapter.export_patterns("./skills_learned.json")
                                                                                                                                                                                                                    ```

                                                                                                                                                                                                                                  ## Contributing New Skills
                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                  To add your own skill to ATLAS:
                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                  1. Create a skill directory in `atlas/skills/[category]/[skill-name]/`
                                                                                                                                                                                                                                  2. 2. Include README.md with:
                                                                                                                                                                                                                                     3.    - Clear purpose statement
                                                                                                                                                                                                                                           -    - Keywords for discovery
                                                                                                                                                                                                                                                -    - Prerequisites and dependencies
                                                                                                                                                                                                                                                   - Usage examples
                                                                                                                                                                                                                                                   -    - Error handling
                                                                                                                                                                                                                                                        - 3. Submit PR to atlas repo
                                                                                                                                                                                                                                                          4. 4. Optionally submit to VoltAgent awesome-agent-skills
                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                             5. See [CONTRIBUTING.md](../CONTRIBUTING.md) for detailed guidelines.
